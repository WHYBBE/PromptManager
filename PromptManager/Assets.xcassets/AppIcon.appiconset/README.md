App icon PNG files should be generated from the SF Symbol `info.circle.text.page.fill`
and saved with these filenames:

- AppIcon-16.png
- AppIcon-32.png
- AppIcon-64.png
- AppIcon-128.png
- AppIcon-256.png
- AppIcon-512.png
- AppIcon-1024.png

You can create them from a vector source or symbol export in a design tool, then place
them in this folder. Xcode will use them as the formal macOS app icon.
